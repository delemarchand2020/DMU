# Compétences des Agents DMU (Skills)

Ce dossier a pour vocation d'héberger les compétences (*skills*) spécifiques destinées aux agents IA (les assistants de codage ou agents métiers virtuels) qui interviendront dans la création et l'exploitation des DMU (Développement en Milieu Utilisateur).

## Objectif

L'objectif est de définir et de structurer les instructions, les comportements, et les règles de gouvernance de ces agents métiers pour garantir qu'ils produisent des solutions :
- **Sécuritaires et conformes** (respect des politiques d'entreprise, absence de fuites de données).
- **Performantes et "Token-efficient"** (pas d'IA superflue au *run* si ce n'est pas nécessaire).
- **Structurées** (respect de gabarits techniques).

Chaque fichier de compétence définira le rôle exact de l'agent dans l'équipe virtuelle (ex: Agent Architecte, Agent Développeur de script de Scraping, Agent Qualité, etc.).
